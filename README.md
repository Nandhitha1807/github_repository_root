# Random Even Number Table

## About

This is a simple Java program that creates a **P × P** table filled with random even numbers between **2 and 16**. The user can enter an even number to highlight, and the program shows how many times that number appears in the table.

## Features

- Generates a random P × P table
- Uses even numbers from 2 to 16
- Highlights the number entered by the user
- Counts how many times the number appears
- Handles invalid inputs

## Technologies Used

- Java
- Scanner
- Random
- 2D Arrays

## How to Run

1. Compile the program:
   ```
   javac Table.java
   ```

2. Run the program:
   ```
   java Table
   ```

3. Enter the array size.

4. Enter an even number between 2 and 16 to highlight.
